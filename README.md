# UAF-Quantum_v_2

**Universal Accounting Field — Quantum Integration (UAF-Q)**

Современная матрёшечная теория времени с явным включением постоянной Планка ℏ.

### Основные идеи
- Три компоненты времени: **t_past** (α), **t_self** (Γ), **t_future**.
- ℏ как квант действия.
- Вывод уравнения Шрёдингера и Lindblad-уравнения как предельных случаев.
- Чётко выверенные размерности во всех уравнениях.

### Файлы
- `theory.md` — полное описание теории
- `equations.tex` — уравнения с размерностями
- `demo.py` — симуляции
