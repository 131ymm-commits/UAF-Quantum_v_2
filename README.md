# UAF-Quantum Model

Реализация модели тройственности времени: **прошлое (α)**, **настоящее (Γ)**, **будущее (β)** для квантовых систем. Модель способна описывать **затухающие когерентные осцилляции** (например, осцилляции Раби) с высокой точностью, но **не предназначена для чисто экспоненциальной релаксации**.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Теоретическая основа

Модель предложена в [репозитории UAF-3.1](https://github.com/131ymm-commits/UAF-3.1) и развивает идею о трёх компонентах времени, связанных нелинейными дифференциальными уравнениями. Ключевая особенность — **комплексное будущее** (`β = β_re + i β_im`), которое позволяет моделировать квантовую интерференцию и когерентные эффекты.
