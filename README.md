# UAF-Quantum (UAF-Q)

**Universal Accounting Field — Quantum Integration**

Современная матрёшечная теория времени, когерентности и стрелки времени, глубоко интегрированная с квантовой механикой.

### Основные идеи
- Время разделено на **t_past** (замороженное), **t_self** (живое настоящее, когерентность) и **t_future** (потенциал).
- Постоянная Планка **ℏ** явно включена как квант действия.
- UAF-Q обобщает уравнение Шрёдингера, Lindblad-уравнение и Page-Wootters механизм emergent time.
- Матрёшечная иерархия уровней объясняет переход от квантового к классическому миру.

### Основные файлы
- `theory.md` — полное теоретическое описание
- `equations.tex` — все ключевые уравнения
- `demo.py` — численные симуляции (двухуровневая система, декогеренция, переход к классике)

### Статус
Активная исследовательская версия. Математика проверена на соответствие стандартной квантовой механике в унитарном и открытом режимах.
