# Математическое описание модели UAF-Quantum

## Переменные состояния

Вектор состояния системы:  
`y = [Γ, α, β_re, β_im, φ]`

| Символ | Описание | Диапазон |
|--------|----------|----------|
| `Γ` | Когерентность настоящего (мера квантовой когерентности) | [0,1] (может кратковременно выходить) |
| `α` | Интеграция прошлого (кристаллизация опыта) | [0,1] |
| `β = β_re + i β_im` | Комплексное будущее (потенциал) | |β|² ≤ 1 |
| `φ` | Квантовая фаза двухуровневой системы | [0, 2π) |

Сумма `α + Γ + |β|²` не сохраняется; она является динамической мерой «полноты времени».

## Уравнения движения

Эволюция определяется системой обыкновенных дифференциальных уравнений:

1. **Изменение когерентности настоящего**  
   \[
   \dot\Gamma = \frac{1}{\hbar}\Bigl[ \eta\,|\beta|\,(1-\Gamma) - \lambda\,\alpha\,\Gamma + \xi\,(-\beta_{\text{im}})\,(1-\Gamma)\,|\beta| \Bigr]
   \]
   - первое слагаемое: усиление когерентности за счёт будущего
   - второе: потеря когерентности из-за кристаллизации прошлого
   - третье: резонансная подкачка от мнимой части будущего (отвечает за осцилляции)

2. **Изменение прошлого (кристаллизация)**  
   \[
   \dot\alpha = \frac{1}{\hbar}\,\lambda\,\Gamma\,(1-\alpha)
   \]

3. **Эволюция комплексного будущего**  
   \[
   \dot\beta_{\text{re}} = -\frac{\eta}{\hbar}\,\beta_{\text{im}}\,\Gamma
   \]  
   \[
   \dot\beta_{\text{im}} = \frac{\eta}{\hbar}\,\beta_{\text{re}}\,\Gamma
   \]  
   Это вращение в комплексной плоскости с частотой, пропорциональной `Γ`.

4. **Изменение фазы**  
   \[
   \dot\varphi = \Delta + \Omega\,\Gamma
   \]

## Наблюдаемая величина

Населённость верхнего уровня двухуровневой системы (вероятность обнаружить систему в возбуждённом состоянии):

\[
P_{\text{верх}} = \frac{1}{2}\Bigl(1 + \Gamma\,\cos\varphi\Bigr)
\]

Это стандартное выражение для матрицы плотности при `Γ = |ρ_{01}|` (когерентность) и `φ = arg(ρ_{01})`.

## Численное решение

Уравнения интегрируются методом `LSODA` (адаптивный, подходит для жёстких систем) из пакета `scipy.integrate.solve_ivp`. Рекомендуемые относительный и абсолютный допуски: `rtol=1e-8, atol=1e-10`.

## Подгонка параметров

Для подгонки под экспериментальные данные используется минимизация среднеквадратичной ошибки (MSE) между моделью и данными. Метод оптимизации: `Nelder-Mead` (симплекс-метод).

Целевая функция:  
\[
\mathcal{L}(\theta) = \frac{1}{N}\sum_{i=1}^N \bigl(P_{\text{модель}}(t_i; \theta) - P_{\text{эксп}}(t_i)\bigr)^2
\]
где `θ = (ħ, Ω, Δ, λ, η, ξ)`.
