# Теория Universal Accounting Field — Quantum (UAF-Q)

## 1. Основные концепции

Время рассматривается как **три компоненты**:
- **t_past** — фиксированное прошлое (α — степень интеграции)
- **t_self** — живое настоящее (Γ — когерентность, суперпозиция)
- **t_future** — потенциальные состояния

**ℏ** — квант действия, задающий масштаб перехода между ними.

## 2. Соответствие стандартной квантовой теории

UAF-Q в соответствующем пределе точно воспроизводит:
- Время-зависимое уравнение Шрёдингера
- Lindblad-уравнение для открытых систем
- Механизм Page-Wootters (реляционное время)

## 3. Ключевые отождествления

- Ψ ≈ √Γ ⋅ exp(iφ) — когерентная амплитуда
- α — параметр декогеренции / интеграции в прошлое
- Φ_{k→l} ⋅ ℰ_{k,l} — матричные элементы взаимодействия
- λ ΔS_env — скорость декогеренции

## 4. Переход к классической физике

При росте α и числа уровней Γ → 0 → система становится классической (траектории + статистическая механика).

**Примечание:** Отождествление "масса = t_past" остаётся гипотезой и требует дальнейшей разработки для согласования с общей теорией относительности.
