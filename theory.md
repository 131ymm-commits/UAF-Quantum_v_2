# UAF-Quantum (UAF-Q) — Теория

## 1. Основные концепции

Время разделено на три компоненты:
- **t_past** — фиксированное прошлое (α_l — степень интеграции, 0..1)
- **t_self** — живое настоящее (Γ_l — когерентность, 0..1)
- **t_future** — потенциал

**ℏ** — фундаментальный квант действия.

## 2. Таблица размерностей

| Величина             | Обозначение     | Размерность       | Примечание                  |
|----------------------|-----------------|-------------------|-----------------------------|
| Γ_l                  | Γ               | безразмерная      | 0 ≤ Γ ≤ 1                   |
| α_l                  | α               | безразмерная      | 0 ≤ α ≤ 1                   |
| τ                    | τ               | [T]               | внутреннее время            |
| ℏ                    | ℏ               | [M L² T⁻¹]        | Дж·с                        |
| E, E_{k,l}           | энергия         | [M L² T⁻²]        |                             |
| Φ_{k→l}              | поток           | [T⁻¹]             | частота                     |
| λ, η, κ, δ, μ        | коэффициенты    | [T⁻¹]             |                             |

## 3. Соответствие стандартной квантовой механике

- α → 0, слабая декогеренция → **уравнение Шрёдингера**
- Полная динамика → **Lindblad-уравнение** для открытых систем

**Примечание:** Идея «масса = замороженное t_past» остаётся гипотезой.
